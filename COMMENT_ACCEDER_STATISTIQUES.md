# 📍 Comment Accéder aux Statistiques ?

## 🎯 Réponse Simple

### Cliquez ici dans le menu :

```
Menu Latéral
  └─ 🏛️ Structure Académique
       └─ 📊 Statistiques ← CLIQUEZ ICI !
```

---

## 📋 Étapes Détaillées

### 1. Ouvrez le Menu
- Cliquez sur l'icône hamburger (☰) si le menu est fermé

### 2. Trouvez "Structure Académique" 🏛️
- C'est dans le menu principal
- Cliquez pour déplier le sous-menu

### 3. Cliquez sur "Statistiques" 📊
- C'est la **dernière entrée** du sous-menu
- Juste après "Spécialités" 🎓

## 🔄 Alternative : URL Directe

Si vous préférez, tapez directement dans la barre d'adresse :

```
/fr/admin/statistics
```

(Remplacez `fr` par votre langue : `en`, `fr`, ou `ar`)

---

## 🐛 Si Vous Ne Voyez Pas l'Entrée

### 1. Rechargez l'Application
```bash
Ctrl + R (Windows/Linux)
Cmd + R (Mac)
```

### 2. Redémarrez le Serveur
```bash
# Arrêtez (Ctrl+C)
# Redémarrez
npm run dev
```

### 3. Vérifiez Votre Rôle
- L'entrée n'est visible que pour **Admin** et **Superadmin**
